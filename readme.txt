A simple plugin to test the WP Color Picker Alpha script

== Description ==

This is just a simple plugin to test the WP Color Picker Alpha script. The WP Color Picker Alpha script can be used to add alpha control to the standard WordPress Color Customizer control


== Installation ==

1. Upload the 'woocommerce-breadcrumbs' folder to your '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the 'Settings > WC Breadcrumbs' menu to update the plugin settings
